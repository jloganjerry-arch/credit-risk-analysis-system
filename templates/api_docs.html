REST API Reference

🔒

All API endpoints are authenticated using an API Key. You must supply the header X-API-KEY on every request.

Your Active API Key: {{ api_key }}

POST /api/v1/predict

Run the Ensemble machine learning pipeline (CatBoost + LightGBM + XGBoost) on an applicant's parameters and record the prediction.

Request JSON Fields

Parameter Type Required Description
contract_type string Yes Type of contract (e.g. "Cash loans", "Revolving loans")
age float Yes Age of the applicant in years
credit float Yes Total credit amount requested in Rs.
income float Yes Total annual income in Rs.
income_type string Yes Income type (e.g. "Working", "Commercial associate")
education string Yes Education level (e.g. "Higher education", "Secondary / special education")
occupation_type string Yes Occupation type (e.g. "Managers", "Core staff", "Laborers")
days_employed float Yes Number of work days employed

Example CURL Request

curl -X POST http://127.0.0.1:5000/api/v1/predict \
  -H "Content-Type: application/json" \
  -H "X-API-KEY: {{ api_key }}" \
  -d '{
    "contract_type": "Cash loans",
    "age": 42,
    "credit": 450000,
    "income": 180000,
    "income_type": "Working",
    "education": "Secondary / special education",
    "occupation_type": "Laborers",
    "days_employed": 1200
  }'

Sample JSON Response

{
  "success": true,
  "transaction_id": 482,
  "risk_score": 14.85,
  "risk_category": "Low Risk",
  "recommendation": "Standard Processing Approved"
}
GET /api/v1/history

Fetch the list of all prediction records ordered from newest to oldest.

Example CURL Request

curl -X GET http://127.0.0.1:5000/api/v1/history \
  -H "X-API-KEY: {{ api_key }}"

Sample JSON Response

{
  "total": 1,
  "predictions": [
    {
      "id": 1,
      "contract_type": "Cash loans",
      "credit_amount": 450000.0,
      "age": 42.0,
      "result": "Low Risk",
      "risk_score": 14.85,
      "occupation_type": "Laborers"
    }
  ]
}
